Basic setup

The first step is to load the library inside your HTML file

Then make sure you have a canvas and paragraph element with these IDs

Then after the body element is loaded you can then call your game.js file on a script element. Or you can call your game.js file inside the onLoad() or $.ready() methods.

To have a game you must at least have these 2 lines:

The first line we are creating a gameGrid object and passing it's X units, Y units and scale values. The X and Y units are how many 1 unit squares you can have horizontally and vertically. The scale is just how many pixels per unit you will have. Ex: X(10) * scale(100) = 1000 pixels horizontally.

In the second line you are just passing our gameGrid object to the createGameLoop function. That's all you need to get a gameGrid running. Now, you can't do much with a blank screen so you probably want to create gameObjects and override some of its methods.

Check out the pong game example file at Github.
