About

Square engine is a group project assignment for Javascript & JQuery class at LWTech. The goal of this project was to create a minimalistic javascript game library that could be used to make simple square games in the spirit of the Atari 2600. Square engine uses the canvas element to generate simple graphics. The library contains various classes to make game developing easier and flexible.
